Install Instructions 
-Install Python3.9.
-Download python install from here
-Run the downloaded file and follow all prompts
-Move the provided QRT .zip files to the desired directory and unzip 
-Open a terminal and navigate to the directory “QRT”
-Run the command “python gui.py 1”
-If prompted, install the required python libraries and Microsoft redistros and try again.


Filenames:
- All necessary files are provided in the QRT.zip file that will be distributed to all customers 
- The file responsible for starting the progam is named 'gui.py' 

Passwords 
- There are no files or systems that require passwords of any sort 

Launch
- The system is launched via the command line using the command 
- "python gui.py 1"
